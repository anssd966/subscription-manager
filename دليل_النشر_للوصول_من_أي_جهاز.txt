بعد النشر، ستحصل على رابط مثل:
https://your-project.vercel.app

يمكنك فتح هذا الرابط من:
- أي كمبيوتر
- أي هاتف
- أي جهاز لوحي
- أي متصفح

البيانات ستكون متزامنة على جميع الأجهزة! ✨

═══════════════════════════════════════════════════════
الطريقة 1: النشر على Vercel (الأسهل والأسرع) ⭐
═══════════════════════════════════════════════════════

الخطوة 1: إنشاء حساب GitHub
─────────────────────────────────────────────

1. اذهب إلى: https://github.com
2. اضغط "Sign up" (سجّل حساب جديد)
3. أكمل التسجيل (مجاني تماماً)

الخطوة 2: رفع المشروع على GitHub
─────────────────────────────────────────────

1. افتح Terminal في مجلد المشروع:
   - اضغط Shift + Right Click في مجلد المشروع
   - اختر "Open PowerShell window here"

2. اكتب الأوامر التالية (استبدل YOUR_USERNAME):

   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/subscription-manager.git
   git push -u origin main

   ⚠️ إذا لم يكن Git مثبتاً:
   - حمّل من: https://git-scm.com/download/win
   - ثبت Git
   - أعد فتح Terminal

الخطوة 3: النشر على Vercel
─────────────────────────────────────────────

1. اذهب إلى: https://vercel.com
2. اضغط "Sign Up" وسجّل بحساب GitHub
3. بعد التسجيل، اضغط "Add New Project"
4. اختر المشروع من قائمة Repositories
5. اضغط "Import"
6. Vercel سيكتشف الإعدادات تلقائياً:
   - Framework: Vite
   - Build Command: npm run build
   - Output Directory: dist
7. اضغط "Deploy"
8. انتظر 1-2 دقيقة
9. ستحصل على رابط مثل: https://subscription-manager.vercel.app

✅ انتهى! الآن يمكنك فتح الرابط من أي جهاز!

═══════════════════════════════════════════════════════
الطريقة 2: النشر عبر Vercel CLI (بدون GitHub)
═══════════════════════════════════════════════════════

1. ثبت Vercel CLI:
   npm install -g vercel

2. في مجلد المشروع:
   vercel

3. اتبع التعليمات:
   - اضغط Enter للاستمرار
   - اختر "Set up and deploy"
   - اضغط Enter لقبول الإعدادات
   - انتظر حتى ينتهي النشر

4. ستحصل على رابط مباشر!

═══════════════════════════════════════════════════════
الطريقة 3: النشر على Netlify (بديل)
═══════════════════════════════════════════════════════

1. اذهب إلى: https://netlify.com
2. سجّل بحساب GitHub
3. اضغط "Add new site" > "Import an existing project"
4. اختر المشروع من GitHub
5. اضغط "Deploy site"
6. انتظر حتى ينتهي النشر
7. ستحصل على رابط مثل: https://your-project.netlify.app

═══════════════════════════════════════════════════════
الطريقة 4: النشر على Firebase Hosting
═══════════════════════════════════════════════════════

1. ثبت Firebase CLI:
   npm install -g firebase-tools

2. سجّل دخول:
   firebase login

3. في مجلد المشروع:
   firebase init hosting

4. اتبع التعليمات:
   - اختر "Use an existing project"
   - اختر مشروعك: subscription-manager-f6570
   - Public directory: dist
   - Single-page app: Yes
   - Set up automatic builds: No

5. بناء المشروع:
   npm run build

6. النشر:
   firebase deploy --only hosting

7. ستحصل على رابط مثل:
   https://subscription-manager-f6570.web.app
