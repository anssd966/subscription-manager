1. ارفع المشروع على GitHub

3. احصل على رابط
4. افتح الرابط من أي جهاز!

═══════════════════════════════════════════════════════
الخطوات التفصيلية:
═══════════════════════════════════════════════════════

الخطوة 1: رفع المشروع على GitHub
─────────────────────────────────────────────

1. اذهب إلى: https://github.com
2. سجّل حساب جديد (مجاني)
3. اضغط "New repository"
4. أدخل اسم: subscription-manager
5. اضغط "Create repository"

6. افتح Terminal في مجلد المشروع
7. اكتب:

   git init
   git add .
   git commit -m "First commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/subscription-manager.git
   git push -u origin main

   (استبدل YOUR_USERNAME باسمك في GitHub)

الخطوة 2: النشر على Vercel
─────────────────────────────────────────────

1. اذهب إلى: https://vercel.com
2. اضغط "Sign Up" وسجّل بحساب GitHub
3. اضغط "Add New Project"
4. اختر المشروع: subscription-manager
5. اضغط "Import"
6. اضغط "Deploy"
7. انتظر 1-2 دقيقة
8. ✅ ستحصل على رابط مثل:
   https://subscription-manager.vercel.app
