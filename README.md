# نظام إدارة الاشتراكات

موقع ويب لإدارة الاشتراكات مع المميزات التالية:

## المميزات

- ✅ إضافة اشتراكات جديدة (اسم الشخص، اسم الاشتراك، تاريخ البداية، المدة)
- ✅ عرض جميع الاشتراكات مع الوقت المتبقي حتى انتهاء كل اشتراك
- ✅ صفحة خاصة لعرض الاشتراكات التي تحتاج إلى تجديد حسب تاريخ محدد أو نطاق زمني
- ✅ تصفية الاشتراكات (الكل، النشطة، القريبة من الانتهاء، المنتهية)
- ✅ واجهة مستخدم عربية جميلة مع دعم RTL
- ✅ حفظ البيانات محلياً في المتصفح
- ✅ **جديد**: تخزين سحابي مع Firebase (اختياري)
- ✅ **جديد**: مزامنة فورية بين الأجهزة
- ✅ **جديد**: جاهز للنشر على الإنترنت

## التثبيت والتشغيل

1. تثبيت المكتبات:
```bash
npm install
```

2. تشغيل المشروع:
```bash
npm run dev
```

3. بناء المشروع للإنتاج:
```bash
npm run build
```

## الصفحات

- **الرئيسية**: عرض جميع الاشتراكات مع الوقت المتبقي
- **إضافة اشتراك**: نموذج لإضافة اشتراك جديد
- **الاشتراكات القادمة للتجديد**: عرض الاشتراكات التي تنتهي في تاريخ محدد أو نطاق زمني

## التقنيات المستخدمة

- React 18
- React Router
- Vite
- Tailwind CSS
- date-fns
- Firebase (للتخزين السحابي)

## إعداد Firebase (اختياري)

لتفعيل التخزين السحابي:

1. اقرأ ملف `إعداد_Firebase.txt` للتعليمات الكاملة
2. أنشئ مشروع Firebase من: https://console.firebase.google.com
3. فعّل Firestore Database
4. انسخ بيانات الإعداد إلى `src/config/firebase.js`

**ملاحظة**: إذا لم تقم بإعداد Firebase، سيستخدم التطبيق localStorage فقط.

## النشر على الإنترنت

لنشر الموقع على الإنترنت:

1. اقرأ ملف `تعليمات_النشر_على_الإنترنت.txt` للتعليمات الكاملة
2. الطريقة الأسهل: ارفع المشروع على GitHub ثم استخدم Vercel
3. أو استخدم Vercel CLI مباشرة

**مواقع النشر المجانية**:
- Vercel (موصى به)
- Netlify
- Firebase Hosting
