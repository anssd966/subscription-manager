═══════════════════════════════════════════════════════
   دليل نشر الموقع على الإنترنت (Vercel)
═══════════════════════════════════════════════════════

الطريقة 1: النشر عبر GitHub (الموصى بها)
─────────────────────────────────────────────

الخطوة 1: إنشاء حساب GitHub
─────────────────────────────

1. اذهب إلى: https://github.com
2. سجّل حساب جديد (مجاني)
3. أكمل التسجيل

الخطوة 2: رفع المشروع إلى GitHub
─────────────────────────────────

1. افتح Terminal في مجلد المشروع
2. اكتب الأوامر التالية:

   git init
   git add .
   git commit -m "Initial commit with Firebase support"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main

   (استبدل YOUR_USERNAME و YOUR_REPO_NAME بقيمك)

3. إذا لم يكن Git مثبتاً:
   - حمّل من: https://git-scm.com/download/win
   - ثبت Git
   - أعد فتح Terminal

الخطوة 3: النشر على Vercel
──────────────────────────────

1. اذهب إلى: https://vercel.com
2. اضغط "Sign Up" وسجّل بحساب GitHub
3. بعد التسجيل، اضغط "Add New Project"
4. اختر المشروع من قائمة Repositories
5. اضغط "Import"
6. Vercel سيكتشف الإعدادات تلقائياً:
   - Framework Preset: Vite
   - Build Command: npm run build
   - Output Directory: dist
7. اضغط "Deploy"
8. انتظر 1-2 دقيقة حتى ينتهي النشر
9. ستحصل على رابط مثل: https://your-project.vercel.app

═══════════════════════════════════════════════════════
الطريقة 2: النشر عبر Vercel CLI
═══════════════════════════════════════════════════════

1. ثبت Vercel CLI:
   npm install -g vercel

2. في مجلد المشروع:
   vercel

3. اتبع التعليمات:
   - اضغط Enter للاستمرار
   - اختر "Set up and deploy"
   - اضغط Enter لقبول الإعدادات الافتراضية
   - انتظر حتى ينتهي النشر

═══════════════════════════════════════════════════════
الخطوة 4: إضافة متغيرات البيئة (اختياري)
═══════════════════════════════════════════════════════

إذا أردت إخفاء بيانات Firebase:

1. في Vercel Dashboard، اذهب إلى Project Settings
2. اختر "Environment Variables"
3. أضف المتغيرات التالية:
   - VITE_FIREBASE_API_KEY
   - VITE_FIREBASE_AUTH_DOMAIN
   - VITE_FIREBASE_PROJECT_ID
   - VITE_FIREBASE_STORAGE_BUCKET
   - VITE_FIREBASE_MESSAGING_SENDER_ID
   - VITE_FIREBASE_APP_ID

4. ثم حدّث src/config/firebase.js لاستخدام:
   apiKey: import.meta.env.VITE_FIREBASE_API_KEY

═══════════════════════════════════════════════════════
الخطوة 5: تحديث تلقائي
═══════════════════════════════════════════════════════

بعد ربط المشروع مع GitHub:
- كل مرة ترفع تغييرات (git push)
- Vercel سينشر التحديثات تلقائياً!
