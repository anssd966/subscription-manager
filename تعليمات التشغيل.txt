هذا مشروع React يحتاج إلى تشغيل خاص

═══════════════════════════════════════════════════════
الخطوة 0: تثبيت Node.js (إذا لم يكن مثبتاً)
═══════════════════════════════════════════════════════

1. اذهب إلى: https://nodejs.org
2. حمّل النسخة LTS (النسخة الموصى بها)
3. ثبتها على جهازك
4. أعد تشغيل الكمبيوتر (أو أعد فتح Terminal)

للتحقق من التثبيت، افتح Terminal واكتب:
    node --version
    npm --version

إذا ظهرت الأرقام، فالتثبيت نجح ✓

═══════════════════════════════════════════════════════
الخطوة 1: تثبيت المكتبات
═══════════════════════════════════════════════════════

1. افتح Terminal أو PowerShell في مجلد المشروع:
   - اضغط Shift + Right Click في مجلد المشروع
   - اختر "Open PowerShell window here"
   
   أو افتح Terminal واكتب:
   cd C:\Users\MSI\Desktop\claent

2. اكتب الأمر التالي واضغط Enter:
   
   npm install

3. انتظر حتى ينتهي التثبيت (قد يستغرق 1-3 دقائق)
   ستظهر رسالة مثل: "added 200 packages"

═══════════════════════════════════════════════════════
الخطوة 2: تشغيل المشروع
═══════════════════════════════════════════════════════

بعد انتهاء التثبيت، اكتب:

   npm run dev

ستظهر رسالة مثل:
   ➜  Local:   http://localhost:5173/
   ➜  Network: use --host to expose

═══════════════════════════════════════════════════════
الخطوة 3: فتح الموقع
═══════════════════════════════════════════════════════

1. افتح المتصفح (Chrome, Edge, Firefox)
2. اذهب إلى العنوان:
   http://localhost:5173

   أو اضغط Ctrl + Click على الرابط الذي يظهر في Terminal

═══════════════════════════════════════════════════════
ملاحظات مهمة:
═══════════════════════════════════════════════════════

✓ لا تغلق Terminal أثناء استخدام الموقع
✓ لإيقاف المشروع: اضغط Ctrl + C في Terminal
✓ البيانات تُحفظ تلقائياً في المتصفح
✓ يمكنك إغلاق المتصفح وإعادة فتحه - البيانات ستبقى
✓ في كل مرة تريد استخدام الموقع، يجب تشغيل: npm run dev

═══════════════════════════════════════════════════════
حل المشاكل:
═══════════════════════════════════════════════════════
 
إذا ظهرت رسالة "npm is not recognized":
→ Node.js غير مثبت أو غير موجود في PATH
→ ثبت Node.js من nodejs.org وأعد تشغيل الكمبيوتر

إذا ظهرت رسالة "port already in use":
→ البورت 5173 مستخدم
→ اكتب: npm run dev -- --port 3000
→ ثم افتح: http://localhost:3000
